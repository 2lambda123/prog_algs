Metrics¶

The metrics module includes tools for calculating various metrics for prognostics results. These metrics are a measurement of the performance of a prognostics algorithms. The analysis functions are described below:

prog_algs.metrics.samples.alpha_lambda(times, toes, ground_truth, lambda_value, alpha, beta)¶

Compute alpha lambda metrics

Parameters

  • times ([float]) – Times corresponding to toes (output from predictors)

  • toes ([float]) – Times of event for a single event, output from predictor

  • ground_truth (float) – Ground Truth time of event for that event

  • lambda_value (float) – lambda

  • alpha (float) – alpha

  • beta (float) – beta

Returns

if alpha-lambda met

Return type

bool

Metrics¶

The metrics module includes tools for calculating various metrics for prognostics results. These metrics are a measurement of the performance of a prognostics algorithms. The analysis functions are described below:

prog_algs.metrics.samples.alpha_lambda(times, eols, ground_truth, lambda_value, alpha, beta)¶

Compute alpha lambda metrics

Parameters

  • times ([float]) – Times corresponding to eols (output from predictors)

  • eols ([float]) – End of life times, output from predictor

  • ground_truth (float) – Ground Truth end of life time

  • lambda_value (fload) – lambda

  • alpha (float) – alpha

  • beta (float) – beta

Returns

if alpha-lambda met

Return type

bool

prog_algs.metrics.samples.eol_metrics(eol, ground_truth=None)¶

Calculate all end of life metrics

Parameters

  • eol ([double]) – End of life times, output from predictor

  • ground_truth (float, optional) – Ground truth end of discharge time. Defaults to None.

Returns

collection of metrics

Return type

dict
